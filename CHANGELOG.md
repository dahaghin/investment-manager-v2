# Changelog

## Beta 1 - 2026-07-17

- Refactored the single-file application into modular CSS and JavaScript assets while preserving the existing UI.
- Added a transaction-based investor ledger with backward compatibility for legacy payment records.
- Implemented monthly compound-profit calculations on active capital plus unpaid accumulated profit.
- Normalized internal date handling to ISO-8601 strings and hardened Jalali edit/display conversion.
- Added investor summary cards and dashboard KPI cards for Beta 1 testing.
